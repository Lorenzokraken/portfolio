## 🚀 Deploy su GitHub Pages

1. Crea la repository su GitHub (es. `vcard-portfolio`)
2. Fai push del progetto:
   ```bash
   git init
   git add .
   git commit -m "First commit"
   git branch -M main
   git remote add origin https://github.com/TUO-NOME/vcard-portfolio.git
   git push -u origin main
Vai su Settings > Pages

In "Build and deployment", seleziona:

Source: Deploy from a branch

Branch: main, / (root)

Attendi il link pubblico es: https://tuo-nome.github.io/vcard-portfolio/
